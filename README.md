# DataHandsOnDataOpsFinOps

Repositório de estudos e práticas de **DataOps** e **FinOps** na **AWS**.

## Sobre
Este projeto reúne materiais e exemplos práticos voltados para:
- organização e automação de entregas de dados (DataOps)
- boas práticas para controle e otimização de custos (FinOps)
- infraestrutura como código e ferramentas de apoio (ex.: Terraform, Airflow, Metabase)

## Conteúdo (alto nível)
- `airflow/` — exemplos e recursos de orquestração
- `terraform/` — infraestrutura como código (IaC)
- `metabase/` — recursos de BI/visualização
- `.github/workflows/` — automações/pipelines (quando aplicável)

## Status
Em evolução (hands-on / aprendizado contínuo).
